## Program Details

### Mystery Science (Elementary — Supplemental)

| Aspect | Detail |
|--------|--------|
| Publisher | Mystery Science (acquired by Discovery Education) |
| Type | Supplemental digital science lessons |
| Grades | K-6 (typically) |
| NGSS Alignment | Partial — supports phenomenon-based learning but not a full NGSS adoption |
| Usage at Fairfax | Listed in LCAP Action 2.5 as a supplemental academic program |
| Budget Line | Part of $724,400 Action 2.5 supplemental programs pool |
| Key Limitation | Mystery Science provides pre-built simulations and video-based phenomena; students observe and interact — they do NOT build their own models |
| ModelIt Relationship | **Complementary** — Mystery Science introduces phenomenon, ModelIt has students model the underlying systems using computational thinking |

### Pitsco Science Lab (Junior High — Electives)

| Aspect | Detail |
|--------|--------|
| Publisher | Pitsco Education |
| Type | Project-based STEM elective lab program |
| Grades | 7–8 (FJH) |
| NGSS Alignment | Limited — engineering/CTE focus; not a comprehensive NGSS SEP curriculum |
| Usage at Fairfax | LCAP Action 3.10 — elective program; 3.5 teacher FTEs dedicated |
| Budget | **$433,000/year** (salary + program costs for 3.5 FTEs) |
| Focus | Engineering design, project-based learning, hands-on STEM electives |
| Key Limitation | Elective-only; does NOT serve all 7-8 students as a required science course; not aligned to CAST science standards tested in Grade 8 |
| ModelIt Relationship | **Complementary** — Pitsco does hands-on engineering design; ModelIt adds NGSS SEP-2 (Developing and Using Models) and SEP-5 (Using Mathematics and Computational Thinking) — the tested science practices on CAST |

## What This Means for ModelIt

### Competitive Positioning

| Current Tool | Strength | Gap ModelIt Fills |
|-------------|---------|------------------|
| Mystery Science | Engaging, phenomenon-based, video content | Students watch pre-built content; do NOT build their own models; no SEP-2 or SEP-5 practice |
| Pitsco Science Lab | Hands-on engineering projects; student engagement | Not NGSS-aligned to CAST standards; elective-only (doesn't serve all students) |
| STEAM Teachers | Dedicated STEM FTE; relationship-building | No confirmed digital modeling platform; no systematic SEP-2/SEP-5 curriculum |
| iReady | Math diagnostic + intervention (strong) | Math-only; no science/NGSS component |
| **ModelIt** | **Students BUILD and RUN their own systems models** | **Directly addresses the SEP-2/SEP-5 gap; bilingual; NSF-funded; complements all existing tools** |

### How ModelIt Complements (Not Competes With) Existing Tools

| Existing Tool | ModelIt Relationship | Talking Point |
|--------------|---------------------|--------------|
| Mystery Science | **Complement** — Mystery Science introduces the phenomenon; ModelIt has students model the underlying system | "Mystery Science helps students notice and wonder. ModelIt has them explain and predict — that's the SEP-2 step that shows up on CAST." |
| Pitsco Science Lab | **Complement** — Pitsco = engineering design; ModelIt = computational modeling and testing | "Pitsco students design things. ModelIt students model the science behind why things work. Together, they get NGSS SEP-3, SEP-2, and SEP-5 covered." |
| iReady | **Cross-tool reinforcement** — Math skills in iReady transfer to quantitative reasoning in ModelIt | "Students using iReady for math intervention use those same math skills when running computational models in ModelIt." |
| STEAM Teachers | **Delivery vehicle** — STEAM teachers can implement ModelIt within their dedicated STEM blocks | "Your STEAM teachers already have the time and the mission. ModelIt gives them a vetted, NSF-backed curriculum tool to deliver NGSS modeling at scale." |
