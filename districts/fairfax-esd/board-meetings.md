## LCAP Budget & Board Calendar

| Event | Typical Timing | Notes |
|-------|---------------|-------|
| LCAP development begins | January/February | Public input period opens |
| LCAP stakeholder meetings | Feb–April | Parent/community advisory meetings |
| LCAP draft presented to board | April/May | Study session presentation |
| LCAP public hearing | May/June | Required before adoption |
| LCAP final adoption | June board meeting (by June 30) | Required for LCFF compliance |
| Budget adoption | June board meeting (by June 30) | Concurrent with LCAP |
| First Interim Financial Report | December | Budget status update |
| Second Interim Financial Report | March | Budget projection |

> **Sales Window**: February–April is the optimal window to introduce ModelIt before LCAP actions are finalized. The June adoption locks in funding allocations for the following year.
