## Vendor Landscape Analysis

### Where ModelIt Fits

ModelIt is a **greenfield addition** to the Fairfax ESD vendor stack. There is no overlapping tool:

| ModelIt Feature | Existing Tool Covering This? | Gap? |
|-----------------|------------------------------|------|
| NGSS SEP-2 (Developing and Using Models) | None — Mystery Science is pre-built; Pitsco is engineering | **YES — gap** |
| NGSS SEP-5 (Math and Computational Thinking) | None confirmed | **YES — gap** |
| Bilingual English/Spanish | None in science | **YES — gap** |
| Student-built systems models | None | **YES — gap** |
| CAST science improvement intervention | None — Mystery Science is supplemental, Pitsco is elective | **YES — gap** |
| Digital platform integrating with Clever SSO | iReady, Lexia, others — ModelIt would follow same pattern | Easy onboarding |

### Warm Intro Paths Through Existing Vendors

| Vendor | Connection to ModelIt | Strategy |
|--------|----------------------|---------|
| **Discovery Education** (owns Mystery Science) | Same science category | No conflict; position as the "next step" after Mystery Science |
| **Curriculum Associates** (iReady) | Math alignment | iReady math skills transfer to ModelIt computational models — cross-sell angle |
| **Clever** | SSO integration | "Already using Clever? ModelIt is Clever-integrated. Zero friction for IT." |
| **Nearpod** | Interactive lesson delivery | Nearpod delivers; ModelIt builds — different use cases; no conflict |

### Competitive Threats

| Competitor | Risk | Mitigation |
|-----------|------|-----------|
| **Amplify Science** | Could land as core K-8 science adoption before ModelIt | Move fast; get in front of Marquez and Clark by March 2026 |
| **OpenSciEd** | Could be adopted as free NGSS curriculum | Differentiate: ModelIt is interactive modeling software, not a textbook replacement |
| **iReady Science** | If Curriculum Associates pitches science expansion | ModelIt complements iReady; different SEP focus (modeling vs. content) |
| **Pitsco expansion** | If Pitsco adds more elective sections | Pitsco is elective-only; ModelIt is NGSS-aligned core supplement |
