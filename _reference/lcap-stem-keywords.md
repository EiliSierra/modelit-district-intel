## Scan Strategy

1. **Full-text search**: Run each keyword against the LCAP document text.
2. **Context extraction**: For each hit, extract the surrounding paragraph and the goal/action number.
3. **Funding identification**: Cross-reference hits with Action Tables to find associated budgets.
4. **Gap analysis**: Note which categories have zero or minimal hits -- these represent potential unmet needs.
5. **Prioritize high-value hits**: `mathematical modeling`, `NGSS`, `phenomenon-based`, `computational thinking`, `science coaching`, and `supplemental curriculum` are the highest-signal keywords for ModelIt alignment.
